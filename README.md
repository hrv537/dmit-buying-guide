## 几个常见坑，顺手说一下

**坑一：流量超限后服务器直接暂停。**

DMIT 流量超了不是降速，是暂停服务直到下月重置。用量大的要提前算好，或者考虑不限流量套餐。不过 T1 系列流量用完后会切换限速继续跑，这点倒是不同。

**坑二：流媒体解锁不稳定。**

DMIT 本身不保证解锁 Netflix、ChatGPT 等平台。很多用户把 DMIT 用作"线路鸡"——跑国内到海外这段连接，另外配一台落地机解锁流媒体。如果你的核心需求是看流媒体，单纯用 DMIT 可能会碰壁。

DMIT 不是那种天天做活动的商家。双十一的特惠机型可能是本季度最低价，等来等去等到的是 Sold Out。
